**Step 1:** Fork and clone this repository.

```bash
  git clone https://github.com/Anugerah20/Uangmu.git
```
**Step 2:** Install node_modules.

```bash
  npm install
```
**Step 3:** Run the app in development mode.

```bash
  npm run dev
```
